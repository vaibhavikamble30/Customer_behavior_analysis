Customer Shopping Behavior Analysis

Overview

This project analyzes customer shopping behavior using Python, MySQL, and Power BI. It covers the complete data analytics workflow—from data cleaning and exploratory data analysis (EDA) to SQL querying and interactive dashboard creation—to generate meaningful business insights.


The dataset contains customer shopping information, including:

* Customer ID
* Age Group
* Gender
* Product Category
* Purchase Amount
* Review Rating
* Subscription Status
* Shipping Type
* Payment Method
* Purchase Frequency


Tools & Technologies

* Python (Pandas, NumPy, Matplotlib)
* Jupyter Notebook
* MySQL
* Power BI
* SQL

Project Workflow

1. Imported the dataset into Jupyter Notebook.
2. Cleaned and preprocessed the data using Python.
3. Performed Exploratory Data Analysis (EDA).
4. Loaded the cleaned data into MySQL.
5. Wrote SQL queries to extract business insights.
6. Built an interactive dashboard in Power BI.


 Dashboard Features

* Total Customers
* Average Purchase Amount
* Average Review Rating
* Revenue by Category
* Sales by Category
* Sales by Age Group
* Subscription Status Analysis
* Interactive filters for Gender, Category, Subscription Status, and Shipping Type



 Key Insights

* Identified top-performing product categories.
* Analyzed customer purchasing patterns.
* Compared subscriber and non-subscriber behavior.
* Evaluated customer satisfaction through review ratings.
* Explored sales trends across different customer segments.

How to Run

1. Clone this repository.
2. Open the Jupyter Notebook and run the data preprocessing steps.
3. Import the cleaned dataset into MySQL.
4. Execute the SQL queries.
5. Open the Power BI (`.pbix`) file to explore the interactive dashboard.
